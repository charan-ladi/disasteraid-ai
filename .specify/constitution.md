# Project Constitution

## Project Name
DisasterAid AI

## Vision

Develop an offline-first, CPU-powered AI platform that converts unstructured disaster reports into structured emergency intelligence to support faster disaster response without requiring internet connectivity.

---

## Core Principles

### 1. CPU First

The application must run entirely on CPU hardware.

GPU acceleration is optional but never required.

---

### 2. Offline First

Core features must function without internet access.

No cloud APIs are permitted.

---

### 3. Open Source

All software components must use Free and Open Source Software.

License:
GNU Affero General Public License v3.0 (AGPL-3.0)

---

### 4. Structured Data

Every input must produce structured output.

Supported formats:

- JSON
- SQLite
- CSV

---

### 5. Privacy

User data never leaves the local machine.

---

### 6. Explainability

AI outputs should be transparent and reviewable.

---

### 7. Simplicity

Prioritize usability for emergency responders under stressful conditions.
